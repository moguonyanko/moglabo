inert属性

input要素やbutton要素などコントロールを行う要素には従来からdisabled属性が使われている。 それ以外の要素が不活性であることを示すのにinert属性が導入された。inert属性がtrueのとき クリックイベントやフォーカスイベントが無視される。 showModal()でダイアログを表示している場合はinert属性に関係なく背景の要素はクリックできない。
