Idle Detection API

IdleDetector

一分間(60000ms)よりも短い時間を指定してstartするとエラーになる。 別スクリーンを操作している場合はuserStateはactiveのままである。 デバイスを全く操作せずにいるとidleになる。放置し続けてもidleが通知されるのは1回だけである。 状態変化なしではchangeイベントが発生しないためである。 画面ロックするとscreenStateはlockedになるがuserStateはactiveになる。 userStateがidleだった場合はidleからactiveに変わる。
