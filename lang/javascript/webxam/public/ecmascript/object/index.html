Object

オブジェクトのシャローコピー

{...obj}

参照はそのままなので元のオブジェクトに変更の影響が及ぶ。

オブジェクトのディープコピー

JSON.parse(JSON.stringify(obj));

関数などクローンできない値はコピーされず破棄される。

structuredClone()

structuredClone(obj);

ディープコピーと同じ結果になるがクローンできない値が含まれるとエラーになる。

groupBy

Object.groupBy(elements, func)

funcの引数に渡した値がgroupByの戻り値に含められる。

Map.groupBy(elements, func)

groupMyの結果をMapで取得したければMap.groupBy()を使う。
